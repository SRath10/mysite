Build Log

This website was built from scratch using simple HTML, CSS, and a bit of JavaScript. I wanted to make something personal instead of using a website builder, mostly just for the fun of it and to see what I could create with code alone. The idea was to keep everything minimal and clean while still adding small details that reflect my interests in technology and operating systems, like the terminal-style elements on the homepage. A lot of the site is intentionally simple, because I wanted to focus on understanding how things actually work rather than relying on heavy frameworks or templates.

While the code was written by me, I’ll admit that the help of large language models was lightly enlisted at times for efficiency — mainly when I got stuck on something small or wanted a quicker way to implement an idea. That said, the overall design, structure, and most of the coding were done by me as part of the process of learning and experimenting. In the end, this site is really just a small project built for fun, curiosity, and the satisfaction of making something myself from the ground up.
